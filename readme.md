# ScratchLab for Moodle 5.2

ScratchLab adds an interactive Scratch 3 experience to Moodle, allowing students to create, save, and load projects directly inside a Moodle activity.

## Key Features

- ✅ Embedded Scratch 3 editor
- 💾 Local project save
- 📂 Local project load
- 🎓 Smooth Moodle activity integration

## Usage

1. Install the activity through the Moodle administrator panel.
2. Add a ScratchLab activity to your course.
3. Students can create and save projects directly inside the activity.

## Why ScratchLab?

ScratchLab makes coding education more accessible and tightly integrates with Moodle. It helps students express ideas quickly through visual block-based programming.

## Requirements

- Moodle 5.2
- Modern web browser with JavaScript enabled
